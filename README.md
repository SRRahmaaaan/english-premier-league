প্রোজেক্ট এর নামঃ English Premier League
<br />
এটা ইংলেন্ড এর ফুটবল লিগ English Premier League এর  উপর ভিত্তি করে বানানো নাম এবং থিম সেখান থেকেই অনুপ্রাণিত ।
<br />
